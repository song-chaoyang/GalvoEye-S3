# GalvoEye-S3 安全说明

> **警告：本设备包含激光模组，使用不当可能造成眼睛和皮肤损伤。使用前请仔细阅读本文档。**

> 最后更新：2026 年 4 月
> 项目：GalvoEye-S3 — 基于 ESP32-S3 的 DIY 激光投影仪

---

## 一、激光安全等级说明

### 1.1 本设备使用的激光等级

本项目使用的激光模组为 **Class IIIa（3R）** 级别，具体参数如下：

| 激光颜色 | 波长 | 功率 | 安全等级 |
|---------|------|------|---------|
| 红色 | 650nm | ≤ 5mW | Class IIIa |
| 绿色 | 520nm | ≤ 5mW | Class IIIa |
| 蓝色 | 450nm | ≤ 5mW | Class IIIa |

### 1.2 Class IIIa 激光的含义

- **Class IIIa（3R）** 激光器的输出功率在 1-5mW 之间
- **直视光束可能造成眼睛损伤**：即使功率只有 5mW，直视激光束也可能在极短时间内对视网膜造成不可逆的损伤
- **皮肤照射风险较低**：5mW 激光对皮肤的直接照射风险较小，但长时间照射仍可能造成损伤
- **反射光也有危险**：镜面反射的激光束同样可能造成眼睛损伤
- **不要低估危险性**：绿色激光（520nm）因为人眼对绿光更敏感，看起来比实际更亮，风险更高

### 1.3 激光安全等级对照表

| 等级 | 功率范围 | 风险等级 | 说明 |
|------|---------|---------|------|
| Class I | < 0.4mW | 安全 | 无危害 |
| Class II | 0.4-1mW | 低风险 | 眨眼反射可保护 |
| Class IIIa | 1-5mW | **中等风险** | **本设备使用此等级** |
| Class IIIb | 5-500mW | 高风险 | 直视和反射均危险 |
| Class IV | > 500mW | 极高风险 | 漫反射也危险 |

## 二、安全保护系统工作原理

本项目设计了**三层安全保护机制**，确保在检测到人体靠近时自动关闭激光。

### 2.1 系统架构

```
┌─────────────────────────────────────────────────┐
│                  安全保护系统                      │
│                                                   │
│  ┌──────────┐  ┌──────────┐  ┌──────────────┐   │
│  │ 第一层   │  │ 第二层   │  │ 第三层       │   │
│  │ VL53L1X  │  │ HC-SR501 │  │ MOSFET      │   │
│  │ ToF 测距 │  │ PIR 感应 │  │ 紧急断电    │   │
│  └────┬─────┘  └────┬─────┘  └──────┬───────┘   │
│       │              │               │           │
│       v              v               v           │
│  ┌─────────────────────────────────────────┐    │
│  │          ESP32-S3 安全逻辑               │    │
│  │  任一层触发 → 激光立即关闭               │    │
│  │  全部安全 → 延时 5 秒后恢复              │    │
│  └─────────────────────────────────────────┘    │
└─────────────────────────────────────────────────┘
```

### 2.2 第一层保护：VL53L1X ToF 测距

- **原理**：使用激光飞行时间（ToF）技术测量前方障碍物距离
- **检测范围**：4cm - 4m
- **触发条件**：当检测到前方 1m 内有障碍物时触发
- **响应时间**：< 50ms
- **恢复条件**：障碍物离开 1.5m 以上，延时 5 秒后恢复
- **优势**：精确测距，可区分人体和其他物体

### 2.3 第二层保护：HC-SR501 人体红外感应

- **原理**：检测人体发出的红外辐射（热辐射）
- **检测范围**：3-7m（可调），锥角约 120 度
- **触发条件**：检测到人体移动时触发
- **响应时间**：< 1s
- **恢复条件**：人体离开检测区域后延时 5 秒恢复
- **优势**：大范围覆盖，作为 ToF 的补充

### 2.4 第三层保护：MOSFET 紧急断电

- **原理**：通过 IRF520 MOSFET 控制激光模组的总电源
- **触发条件**：
  - 软件检测到前两层保护触发
  - Web 界面紧急停止按钮
  - 串口紧急停止命令
  - ESP32 系统异常（看门狗复位时默认关闭）
- **响应时间**：< 10ms（硬件级断电）
- **恢复条件**：需要手动通过 Web 界面或串口发送恢复命令
- **优势**：即使软件崩溃，也能通过硬件断电保护

### 2.5 故障安全设计（Fail-Safe）

- 所有安全传感器默认状态为"触发"（检测到危险）
- ESP32 重启时，MOSFET 默认关闭（激光断电）
- I2C 通信中断时，ToF 传感器视为触发状态
- 任何一层保护失效时，其他层仍可独立工作

## 三、使用注意事项

### 3.1 通用安全规则

1. **永远不要直视激光束**：即使功率只有 5mW，也可能造成视网膜损伤
2. **永远不要将激光指向他人**：这是违法行为，可能造成严重伤害
3. **永远不要将激光指向车辆或飞机**：可能导致交通事故，属于违法行为
4. **佩戴激光防护眼镜**：操作激光设备时务必佩戴对应波长的防护眼镜
   - 红色激光：OD4+ @ 650nm
   - 绿色激光：OD4+ @ 520nm（最重要，人眼对绿光最敏感）
   - 蓝色激光：OD4+ @ 450nm
   - 建议购买全波段防护眼镜（OD4+ @ 450-650nm）
5. **不要使用光学仪器观察激光**：望远镜、显微镜、放大镜等会聚焦激光，大幅增加危险性
6. **远离易燃物品**：激光可能点燃深色或易燃材料
7. **不要自行改装激光模组**：增加功率会使设备升级到更高危险等级

### 3.2 安装和使用环境

1. **投影面选择**：使用白色或浅色漫反射表面作为投影面，避免镜面或玻璃
2. **安装高度**：建议安装在儿童无法触及的高度（> 1.5m）
3. **投影区域**：确保激光投影区域内没有人员活动
4. **环境光线**：在较暗的环境中使用效果更好，但也要注意安全
5. **通风**：确保使用环境通风良好

### 3.3 操作流程

1. **开机前**：
   - 检查所有安全传感器是否正常工作
   - 确认投影区域内无人员
   - 佩戴激光防护眼镜
2. **运行中**：
   - 不要将任何身体部位伸入投影区域
   - 保持安全传感器清洁、无遮挡
   - 定期检查安全系统是否正常
3. **关机后**：
   - 等待激光模组完全冷却
   - 拔掉电源插头
   - 将设备存放在儿童无法触及的地方

### 3.4 维护注意事项

1. 定期清洁 VL53L1X 传感器窗口（灰尘会影响测距精度）
2. 定期检查 HC-SR501 的检测范围是否正常
3. 定期检查 MOSFET 模块是否正常开关
4. 如发现安全系统异常，立即停止使用并维修
5. 不要在安全系统故障的情况下使用设备

## 四、儿童安全特别提醒

### 4.1 儿童面临的特殊风险

- 儿童的好奇心强，可能直视激光束
- 儿童的晶状体更透明，激光更容易到达视网膜
- 儿童的瞳孔更大，允许更多激光进入眼睛
- 儿童缺乏危险判断能力，不会主动避开激光

### 4.2 儿童安全措施

1. **设备必须安装在儿童无法触及的位置**
2. **不要让儿童操作此设备**
3. **不要在儿童活动的房间使用此设备**
4. **存放时确保儿童无法接触到设备**
5. **教育年龄较大的儿童激光的危险性**
6. **如果儿童不慎暴露在激光下，立即就医检查**

## 五、法律法规提醒

### 5.1 中国法律法规

1. **《中华人民共和国激光产品安全辐射标准》**（GB 7247.1）：规定了激光产品的安全要求和分类
2. **《中华人民共和国治安管理处罚法》**：故意使用激光照射他人可能构成违法行为
3. **《中华人民共和国民用航空法》**：向航空器发射激光属于违法行为，可追究刑事责任
4. **《中华人民共和国道路交通安全法》**：向车辆发射激光干扰驾驶属于违法行为

### 5.2 法律红线

| 行为 | 后果 |
|------|------|
| 用激光照射他人眼睛 | 故意伤害罪，可处三年以下有期徒刑 |
| 用激光照射飞机 | 危害公共安全罪，可处五年以上有期徒刑 |
| 用激光照射车辆 | 危害公共安全罪，可追究刑事责任 |
| 在公共场所无防护使用激光 | 可能违反治安管理条例 |
| 销售无安全标识的激光产品 | 违反产品质量法 |

## 六、紧急处理

### 6.1 激光照射眼睛

1. **立即移开视线**，不要揉眼睛
2. **闭眼休息**，不要强光刺激
3. **立即就医**，告知医生激光波长和照射时间
4. 即使当时没有不适，也建议进行眼科检查（有些损伤有延迟效应）

### 6.2 激光照射皮肤

1. **立即移开激光**
2. **用冷水冲洗**照射部位
3. 如有红肿或疼痛，**就医检查**

### 6.3 设备故障

1. **立即断电**
2. **不要尝试在通电状态下检修**
3. 检查安全系统是否正常
4. 排除故障后再重新使用
