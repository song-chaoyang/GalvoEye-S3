# GalvoEye-S3 外壳设计文档

> 最后更新：2026 年 4 月
> 项目：GalvoEye-S3 -- 基于 ESP32-S3 的 DIY 激光投影仪

---

## 一、外壳设计要求

### 1.1 功能性开孔要求

外壳需要为以下部件预留开孔或窗口：

| 开孔位置 | 尺寸参考 | 用途 | 备注 |
|---------|---------|------|------|
| 激光出光口 | 约 10mm 圆孔 | RGB 激光束射出 | **必须安装激光安全窗片** |
| 摄像头窗口 | 约 8mm x 8mm 方孔 | OV2640 摄像头视野 | 需要透明亚克力或玻璃窗片 |
| ToF 传感器窗口 | 约 6mm 圆孔 | VL53L1X 测距传感器 | 需要透明窗口，不能遮挡 |
| PIR 传感器窗口 | 约 10mm 圆孔（菲涅尔透镜） | HC-SR501 人体感应 | 白色半球形菲涅尔透镜 |
| 接线出口 | 约 20mm x 10mm 矩形 | 电源线、杜邦线引出 | 可用橡胶垫圈密封 |
| 散热孔 | 若干 3-5mm 圆孔 | ESP32-S3 和稳压模块散热 | 建议在顶部和侧面开孔 |
| SD 卡槽口 | 约 12mm x 2mm 槽孔 | MicroSD 卡插拔 | 侧面开槽 |
| 复位/BOOT 按钮 | 2 个约 4mm 圆孔 | ESP32-S3 复位和下载模式 | 侧面开孔 |

### 1.2 散热要求

- ESP32-S3 在运行 WiFi 和摄像头时功耗约 300-500mA，会产生一定热量
- 振镜驱动板工作时发热量较大
- 建议在外壳顶部和侧面设置散热通风孔
- 如果使用 PLA 材料，建议在 ESP32-S3 下方增加散热铜柱或散热片

### 1.3 安全要求

- 激光出光口 **必须** 安装激光安全窗片（OD4+，450-650nm 全波段防护）
- 外壳应能承受意外跌落，内部元件不会松动
- 所有金属螺丝不应接触到面包板上的带电部分
- 外壳材料应具备一定的阻燃性

## 二、方案 A：面包板方案外壳（推荐新手）

### 2.1 概述

面包板方案使用 830 孔面包板（约 165 x 55 x 10mm）搭建电路，外壳需要足够大以容纳面包板、ESP32-S3-CAM 模块以及所有外接模块。

3D 打印外壳是最灵活的方案，可以精确控制开孔位置和尺寸。推荐使用 OpenSCAD 进行参数化设计，方便根据实际元件位置调整。

### 2.2 尺寸

- 外部尺寸：**180mm x 70mm x 50mm**（长 x 宽 x 高）
- 壁厚：2mm
- 内部空间：约 176mm x 66mm x 48mm

> **说明：** 830 孔面包板尺寸约 165 x 55 x 10mm，加上 ESP32-S3-CAM 模块和杜邦线高度，需要 50mm 高度。180mm 长度留出面包板两侧的模块（ToF 传感器、PIR 传感器、按钮等）空间。70mm 宽度留出面包板两侧的杜邦线和模块空间。

### 2.3 元件布局建议

```
外壳俯视图（180mm x 70mm）

  ┌──────────────────────────────────────────────────────────────┐
  │                                                              │
  │  [ToF]  ┌─────────────────────────────────────┐  [PIR]     │
  │  传感器 │                                     │  传感器     │
  │         │         830 孔面包板                 │            │
  │  [按钮] │         (165 x 55mm)                │  [MOSFET]  │
  │  x3     │                                     │  模块      │
  │         │    [MCP4822]  [LM324]               │            │
  │         │                                     │            │
  │         └─────────────────────────────────────┘            │
  │                                                              │
  │  [ESP32-S3-CAM] -- 通过杜邦线连接到面包板                    │
  │                                                              │
  └──────────────────────────────────────────────────────────────┘

  前面板：激光出光口 + 摄像头窗口 + ToF 窗口
  后面板：接线出口 + 电源接口
  左侧板：SD 卡槽 + 按钮 + 散热孔
  右侧板：PIR 窗口 + 散热孔
  顶板：散热孔阵列
  底板：面包板固定槽
```

**布局要点：**

1. 面包板放在外壳中央，用双面胶或卡槽固定
2. ESP32-S3-CAM 放在面包板旁边，通过杜邦线连接
3. ToF 传感器靠近前面板（需要窗口对准投影方向）
4. PIR 传感器靠近前面板侧面（需要菲涅尔透镜窗口）
5. 按钮放在左侧板附近（方便手指操作）
6. MOSFET 模块放在右侧，靠近激光模组电源线出口
7. 振镜驱动板可以放在面包板上或单独固定

### 2.4 材料选择

| 材料 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **PLA** | 打印容易，表面光滑，价格低 | 耐热性差（60°C 变形），不适合高温环境 | 新手入门 |
| **PETG** | 耐热性好（80°C），有一定韧性 | 打印略有翘边，表面不如 PLA 光滑 | **推荐** |
| ABS | 耐热性好，强度高 | 打印需要封闭舱，有异味 | 高级用户 |

> **建议：** 如果 ESP32-S3 发热明显，优先选择 PETG 材料。

### 2.5 OpenSCAD 参数化设计代码

以下提供两种方案的外壳设计代码，通过修改变量即可切换。

```openscad
// ============================================================
// GalvoEye-S3 外壳 - OpenSCAD 参数化设计
// ============================================================
// 使用方法：复制到 https://openscad.cloud/ 或本地 OpenSCAD 中渲染
// 通过修改 plan 变量选择方案：plan = "breadboard" 或 "perfboard"
// ============================================================

// --- 选择方案 ---
// "breadboard" = 方案 A（面包板，180x70x50mm）
// "perfboard"  = 方案 B（万能板焊接，100x65x45mm）
plan = "breadboard";

// --- 根据方案自动设置尺寸 ---
if (plan == "breadboard") {
    length = 180;       // 长度（mm）
    width = 70;         // 宽度（mm）
    height = 50;        // 高度（mm）
} else {
    length = 100;       // 长度（mm）
    width = 65;         // 宽度（mm）
    height = 45;        // 高度（mm）
}

wall = 2;          // 壁厚（mm）

// --- 激光出光口 ---
laser_x = length/2;       // 出光口 X 位置（前端中心）
laser_y = 0;              // 出光口 Y 位置
laser_dia = 10;           // 出光口直径（mm）

// --- 摄像头窗口 ---
cam_x = length/2;         // 摄像头 X 位置（前端）
cam_y = 15;               // 摄像头 Y 位置（偏上）
cam_w = 8;                // 窗口宽度
cam_h = 8;                // 窗口高度

// --- ToF 传感器窗口 ---
tof_x = length/2;         // ToF X 位置（前端）
tof_y = -15;              // ToF Y 位置（偏下）
tof_dia = 6;              // 窗口直径

// --- PIR 传感器窗口 ---
pir_x = 10;               // PIR X 位置（侧面）
pir_y = 0;                // PIR Y 位置
pir_dia = 12;             // 菲涅尔透镜直径

// --- 接线出口 ---
cable_x = 0;              // 接线口 X 位置（后端）
cable_y = 0;              // 接线口 Y 位置
cable_w = 20;             // 接线口宽度
cable_h = 10;             // 接线口高度

// --- SD 卡槽 ---
sd_x = 0;                 // SD 卡槽 X 位置（侧面）
sd_y = -10;               // SD 卡槽 Y 位置
sd_w = 14;                // 槽口宽度
sd_h = 3;                 // 槽口高度

// --- 按钮 ---
btn1_x = 0;               // 按钮 1 X 位置（侧面）
btn1_y = 10;              // 按钮 1 Y 位置
btn_dia = 4;              // 按钮开孔直径

btn2_x = 0;               // 按钮 2 X 位置
btn2_y = 0;               // 按钮 2 Y 位置

// --- 散热孔 ---
vent_dia = 3;             // 散热孔直径
vent_spacing = 8;         // 散热孔间距

// --- 安装孔 ---
hole_dia = 2.5;           // 安装孔直径
hole_offset = 3;          // 安装孔距边缘偏移

// --- 固定铜柱高度 ---
pillar_height = 6;        // 铜柱高度（mm）
pillar_dia = 4;           // 铜柱直径

$fn = 64;                 // 圆弧精度

// ============================================================
// 主体模型
// ============================================================

difference() {
    // 外壳主体
    cube([length, width, height], center=true);

    // 内部空腔
    translate([0, 0, wall/2])
        cube([length - 2*wall, width - 2*wall, height - wall], center=true);

    // --- 前面板开孔 ---

    // 激光出光口
    translate([laser_x - length/2, laser_y, 0])
        cylinder(d=laser_dia, h=wall*2+1, center=true);

    // 摄像头窗口
    translate([laser_x - length/2, cam_y, 0])
        cube([cam_w, cam_h, wall*2+1], center=true);

    // ToF 传感器窗口
    translate([tof_x - length/2, tof_y, 0])
        cylinder(d=tof_dia, h=wall*2+1, center=true);

    // --- 后面板开孔 ---

    // 接线出口
    translate([-length/2, cable_y, -height/4])
        cube([wall*2+1, cable_w, cable_h], center=true);

    // --- 侧面开孔（Y 方向两侧） ---

    // PIR 传感器窗口（右侧）
    translate([pir_x, width/2, 0])
        cylinder(d=pir_dia, h=wall*2+1, center=true);

    // SD 卡槽（左侧）
    translate([sd_x, -width/2, sd_y])
        cube([sd_w, wall*2+1, sd_h], center=true);

    // 按钮 1（左侧）
    translate([btn1_x, -width/2, btn1_y])
        cylinder(d=btn_dia, h=wall*2+1, center=true);

    // 按钮 2（左侧）
    translate([btn2_x, -width/2, btn2_y])
        cylinder(d=btn_dia, h=wall*2+1, center=true);

    // --- 顶面散热孔 ---
    // 根据外壳尺寸动态计算散热孔数量
    vent_cols = floor((length - 40) / vent_spacing);
    vent_rows = floor((width - 20) / vent_spacing);

    for (i = [0:vent_cols-1]) {
        for (j = [0:vent_rows-1]) {
            translate([
                -vent_cols/2 * vent_spacing + i * vent_spacing + vent_spacing/2,
                -vent_rows/2 * vent_spacing + j * vent_spacing + vent_spacing/2,
                height/2
            ])
                cylinder(d=vent_dia, h=wall*2+1, center=true);
        }
    }

    // --- 底面散热孔 ---
    for (i = [0:vent_cols-1]) {
        translate([
            -vent_cols/2 * vent_spacing + i * vent_spacing + vent_spacing/2,
            0,
            -height/2
        ])
            cylinder(d=vent_dia, h=wall*2+1, center=true);
    }

    // --- 安装孔（底部） ---
    // 四角安装孔
    pcb_l = length - 2*wall - 2*hole_offset;
    pcb_w = width - 2*wall - 2*hole_offset;

    for (sx = [-1, 1]) {
        for (sy = [-1, 1]) {
            translate([
                sx * pcb_l/2,
                sy * pcb_w/2,
                -height/2 + wall/2
            ])
            cylinder(d=hole_dia, h=wall*2, center=true);
        }
    }
}

// ============================================================
// 固定铜柱（可选，打印时取消注释）
// ============================================================
/*
// 四个铜柱
for (sx = [-1, 1]) {
    for (sy = [-1, 1]) {
        translate([
            sx * pcb_l/2,
            sy * pcb_w/2,
            -height/2 + wall + pillar_height/2
        ])
        difference() {
            cylinder(d=pillar_dia, h=pillar_height, center=true);
            cylinder(d=hole_dia, h=pillar_height+1, center=true);
        }
    }
}
*/
```

### 2.6 打印参数建议

| 参数 | PLA | PETG |
|------|-----|------|
| 层高 | 0.2mm | 0.2mm |
| 填充率 | 20-30% | 20-30% |
| 壳层数 | 3 层 | 3 层 |
| 打印速度 | 50-60 mm/s | 40-50 mm/s |
| 喷嘴温度 | 200-210°C | 230-245°C |
| 热床温度 | 60°C | 70-80°C |
| 是否需要支撑 | 否（设计已考虑） | 否 |
| 冷却风扇 | 100% | 30-50% |

### 2.7 3D 打印床要求

| 方案 | 最大尺寸 | 最小打印床要求 |
|------|---------|--------------|
| 方案 A（面包板） | 180 x 70 x 50mm | **200mm x 200mm**（推荐） |
| 方案 B（万能板） | 100 x 65 x 45mm | 120mm x 120mm |

> **重要：** 方案 A 的面包板外壳长度为 180mm，需要至少 200mm 的 3D 打印床。常见的 Ender 3 打印床为 220 x 220mm，可以满足要求。如果打印床不够大，请选择方案 B（万能板焊接方案）。

### 2.8 打印后处理

1. 去除支撑材料（如果有）
2. 用砂纸打磨开孔边缘，确保光滑无毛刺
3. 在激光出光口安装安全窗片（见安全设计章节）
4. 在摄像头窗口粘贴透明亚克力片或玻璃片
5. 用双面胶将面包板固定在外壳底部

## 三、方案 B：万能板焊接方案外壳（更紧凑）

### 3.1 概述

如果选择万能板（洞洞板）焊接方案，元件尺寸更小，可以使用更紧凑的外壳。焊接后元件高度更低，整体体积更小。

### 3.2 尺寸

- 外部尺寸：**100mm x 65mm x 45mm**（长 x 宽 x 高）
- 壁厚：2mm
- 内部空间：约 96mm x 61mm x 43mm

### 3.3 元件布局建议

```
外壳俯视图（100mm x 65mm）

  ┌──────────────────────────────────────┐
  │                                      │
  │  [ToF]  ┌──────────────────┐  [PIR] │
  │         │                  │         │
  │  [按钮] │   万能板/PCB     │         │
  │  x3     │   (70 x 50mm)   │ [MOSFET]│
  │         │                  │         │
  │         │  [MCP4822][LM324]│         │
  │         └──────────────────┘         │
  │                                      │
  │  [ESP32-S3-CAM] -- 焊接/排针连接    │
  │                                      │
  └──────────────────────────────────────┘
```

**布局要点：**

1. 万能板放在外壳中央，用铜柱固定
2. ESP32-S3-CAM 通过排母焊接在万能板上
3. ToF 传感器和 PIR 传感器靠近前面板
4. 按钮放在左侧板附近
5. MOSFET 模块放在右侧

### 3.4 材料选择

与方案 A 相同，推荐 PETG 材料。

### 3.5 OpenSCAD 代码

使用第二章 2.5 节中的代码，将 `plan` 变量修改为 `"perfboard"` 即可自动切换到方案 B 的尺寸。

### 3.6 打印参数

与方案 A 相同（见 2.6 节）。

### 3.7 打印后处理

1. 去除支撑材料（如果有）
2. 用砂纸打磨开孔边缘
3. 安装安全窗片和摄像头窗片
4. 用 M2 铜柱和螺丝固定万能板

## 四、方案 C：亚克力激光切割外壳

### 4.1 概述

亚克力激光切割外壳外观整洁、制作快速，适合有一定动手能力的用户。可以通过淘宝定制切割服务。

### 4.2 材料规格

- 材料：**3mm 黑色透明亚克力板**
- 优点：外观专业，激光切割边缘光滑
- 缺点：开孔位置固定后不易修改，散热较差

### 4.3 板材尺寸

#### 方案 A（面包板）板材

共需要 6 块板材：

| 板材 | 尺寸（mm） | 数量 | 说明 |
|------|-----------|------|------|
| 顶板 | 180 x 70 | 1 | 顶面，带散热孔阵列 |
| 底板 | 180 x 70 | 1 | 底面，带安装孔 |
| 前面板 | 70 x 50 | 1 | 激光出光口 + 摄像头窗口 + ToF 窗口 |
| 后面板 | 70 x 50 | 1 | 接线出口 |
| 左侧板 | 180 x 50 | 1 | SD 卡槽 + 按钮 + 散热孔 |
| 右侧板 | 180 x 50 | 1 | PIR 传感器窗口 + 散热孔 |

#### 方案 B（万能板焊接）板材

共需要 6 块板材：

| 板材 | 尺寸（mm） | 数量 | 说明 |
|------|-----------|------|------|
| 顶板 | 100 x 65 | 1 | 顶面，带散热孔阵列 |
| 底板 | 100 x 65 | 1 | 底面，带安装孔 |
| 前面板 | 65 x 45 | 1 | 激光出光口 + 摄像头窗口 + ToF 窗口 |
| 后面板 | 65 x 45 | 1 | 接线出口 |
| 左侧板 | 100 x 45 | 1 | SD 卡槽 + 按钮 + 散热孔 |
| 右侧板 | 100 x 45 | 1 | PIR 传感器窗口 + 散热孔 |

### 4.4 各板开孔描述

#### 顶板（方案 A：180 x 70mm / 方案 B：100 x 65mm）


```
激光出光口：中心偏下，直径 10mm 圆孔

ToF 传感器窗口：中心偏下偏左，直径 6mm 圆孔
```

#### 后面板（方案 A：70 x 50mm / 方案 B：65 x 45mm）

```

```

### 4.6 组装方式

- 使用 M2 x 8mm 螺丝 + M2 螺母在角部连接各板
- 板材交接处使用 3mm 亚克力角件加固
- 面包板使用双面胶固定在底板上
- 万能板使用 M2 铜柱固定在底板上

## 五、外壳组装步骤（通用）

无论选择哪种方案，组装流程基本一致：

### 5.1 准备工作

1. 确认所有元件已搭建/焊接完成并通过测试
2. 确认面包板/万能板尺寸与外壳内部空间匹配
3. 准备固定螺丝、铜柱、螺母

### 5.2 安装顺序

1. **安装固定铜柱**：在外壳底板上安装 4 根铜柱
2. **放置面包板/万能板**：将面包板对准铜柱位置放入（面包板用双面胶固定，万能板用螺丝固定）
3. **连接外部线缆**：连接电源线、传感器线缆等
4. **安装顶板**：盖上顶板，用螺丝固定
5. **安装安全窗片**：在激光出光口安装激光安全窗片
6. **安装摄像头窗片**：在摄像头窗口粘贴透明窗片
7. **最终检查**：确认所有开孔对齐，无遮挡

### 5.3 注意事项

- 安装前确认面包板/万能板上无短路
- 螺丝不要拧太紧，避免压坏面包板
- 激光出光口的安全窗片必须安装到位
- 确保摄像头窗口干净无划痕
- 面包板方案的外壳较大，注意 3D 打印床是否足够

## 六、散热注意事项

### 6.1 主要发热元件

| 元件 | 典型功耗 | 散热建议 |
|------|---------|---------|
| ESP32-S3-CAM | 300-500mA @ 3.3V | 顶部散热孔 + PCB 铜皮散热 |
| 振镜驱动板 | 50-100mA @ 5V | 侧面散热孔 |
| MCP4822 + LM324 | < 50mA | 通常不需要额外散热 |

### 6.2 散热优化建议

1. **通风孔设计**：在顶部和侧面设置通风孔，利用自然对流散热
2. **热隔离**：将发热量大的元件与温度敏感元件（ToF 传感器）分开布置
3. **导热铜柱**：使用铜柱固定万能板，铜柱可以辅助导热到底板
4. **避免密闭**：不要将外壳完全密封，至少保留 20% 的表面积作为通风孔
5. **环境温度**：避免在环境温度超过 35°C 的地方使用

## 七、激光出光口安全设计

### 7.1 安全窗片要求

激光出光口 **必须** 安装激光安全窗片，这是防止意外激光泄露的关键安全措施。

| 参数 | 要求 |
|------|------|
| 光学密度（OD） | OD4+（衰减 10000 倍以上） |
| 波长范围 | 450nm - 650nm（覆盖 RGB 三色激光） |
| 透过率 | 对可见光有一定透过率（方便观察） |
| 材料 | 聚碳酸酯（PC）或专用激光安全玻璃 |
| 安装方式 | 用胶水或压圈固定在出光口 |

### 7.2 安全窗片购买建议

```
淘宝搜索关键词：
- 激光安全窗片 OD4
- 激光防护片 450-650nm
- 激光眼镜片 OD4 全波段
```

> **注意：** 安全窗片的 OD 值必须与激光功率匹配。本项目使用 5mW 激光，OD4 窗片可将泄露激光衰减到 0.5μW 以下，远低于安全限值。

### 7.3 安装方法

1. 将安全窗片裁剪为略大于出光口的圆形（直径约 12mm）
2. 在出光口内侧涂少量 UV 胶或环氧树脂
3. 将窗片压入出光口，确保平整无气泡
4. 等待胶水完全固化
5. 用激光功率计验证窗片的衰减效果

### 7.4 安全窗片维护

- 定期检查窗片是否有裂纹或脱胶
- 如发现窗片损坏，立即更换
- 清洁时使用柔软的无尘布，避免刮花
- 不要使用酒精等溶剂清洁聚碳酸酯窗片
