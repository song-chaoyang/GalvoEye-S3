# GalvoEye-S3 外壳设计文档

> 最后更新：2026 年 4 月
> 项目：GalvoEye-S3 — 基于 ESP32-S3 的 DIY 激光投影仪

---

## 一、外壳设计要求

### 1.1 功能性开孔要求

外壳需要为以下部件预留开孔或窗口：

| 开孔位置 | 尺寸参考 | 用途 | 备注 |
|---------|---------|------|------|
| 激光出光口 | 约 10mm 圆孔 | RGB 激光束射出 | **必须安装激光安全窗片** |
| 摄像头窗口 | 约 8mm x 8mm 方孔 | OV2640 摄像头视野 | 需要透明亚克力或玻璃窗片 |
| ToF 传感器窗口 | 约 6mm 圆孔 | VL53L1X 测距传感器 | 需要透明窗口，不能遮挡 |
| PIR 传感器窗口 | 约 10mm 圆孔（菲涅尔透镜） | HC-SR501 人体感应 | 白色半球形菲涅尔透镜 |
| 接线出口 | 约 20mm x 10mm 矩形 | 电源线、杜邦线引出 | 可用橡胶垫圈密封 |
| 散热孔 | 若干 3-5mm 圆孔 | ESP32-S3 和稳压模块散热 | 建议在顶部和侧面开孔 |
| SD 卡槽口 | 约 12mm x 2mm 槽孔 | MicroSD 卡插拔 | 侧面开槽 |
| 复位/BOOT 按钮 | 2 个约 4mm 圆孔 | ESP32-S3 复位和下载模式 | 侧面开孔 |

### 1.2 散热要求

- ESP32-S3 在运行 WiFi 和摄像头时功耗约 300-500mA，会产生一定热量
- AMS1117-3.3 稳压模块在 5V 输入时也会发热
- 振镜驱动板工作时发热量较大
- 建议在外壳顶部和侧面设置散热通风孔
- 如果使用 PLA 材料，建议在 ESP32-S3 和稳压模块下方增加散热铜柱或散热片

### 1.3 安全要求

- 激光出光口 **必须** 安装激光安全窗片（OD4+，450-650nm 全波段防护）
- 外壳应能承受意外跌落，内部元件不会松动
- 所有金属螺丝不应接触到 PCB 上的带电部分
- 外壳材料应具备一定的阻燃性

## 二、方案 A：3D 打印外壳（推荐新手）

### 2.1 概述

3D 打印外壳是最灵活的方案，可以精确控制开孔位置和尺寸。推荐使用 OpenSCAD 进行参数化设计，方便根据实际元件位置调整。

### 2.2 尺寸

- 外部尺寸：约 **80mm x 60mm x 40mm**（长 x 宽 x 高）
- 壁厚：2mm
- 内部空间：约 76mm x 56mm x 36mm

### 2.3 材料选择

| 材料 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **PLA** | 打印容易，表面光滑，价格低 | 耐热性差（60°C 变形），不适合高温环境 | 新手入门 |
| **PETG** | 耐热性好（80°C），有一定韧性 | 打印略有翘边，表面不如 PLA 光滑 | **推荐** |
| ABS | 耐热性好，强度高 | 打印需要封闭舱，有异味 | 高级用户 |

> **建议：** 如果 ESP32-S3 发热明显，优先选择 PETG 材料。

### 2.4 OpenSCAD 参数化设计代码

以下是一个简单版本的外壳设计，包含所有必要的开孔：

```openscad
// ============================================================
// GalvoEye-S3 外壳 - OpenSCAD 参数化设计
// ============================================================
// 使用方法：复制到 https://openscad.cloud/ 或本地 OpenSCAD 中渲染
// ============================================================

// --- 外壳总体尺寸 ---
length = 80;       // 长度（mm）
width = 60;        // 宽度（mm）
height = 40;       // 高度（mm）
wall = 2;          // 壁厚（mm）

// --- 激光出光口 ---
laser_x = length/2;       // 出光口 X 位置（前端中心）
laser_y = 0;              // 出光口 Y 位置
laser_dia = 10;           // 出光口直径（mm）

// --- 摄像头窗口 ---
cam_x = length/2;         // 摄像头 X 位置（前端）
cam_y = 15;               // 摄像头 Y 位置（偏上）
cam_w = 8;                // 窗口宽度
cam_h = 8;                // 窗口高度

// --- ToF 传感器窗口 ---
tof_x = length/2;         // ToF X 位置（前端）
tof_y = -15;              // ToF Y 位置（偏下）
tof_dia = 6;              // 窗口直径

// --- PIR 传感器窗口 ---
pir_x = 10;               // PIR X 位置（侧面）
pir_y = 0;                // PIR Y 位置
pir_dia = 12;             // 菲涅尔透镜直径

// --- 接线出口 ---
cable_x = 0;              // 接线口 X 位置（后端）
cable_y = 0;              // 接线口 Y 位置
cable_w = 20;             // 接线口宽度
cable_h = 10;             // 接线口高度

// --- SD 卡槽 ---
sd_x = 0;                 // SD 卡槽 X 位置（侧面）
sd_y = -10;               // SD 卡槽 Y 位置
sd_w = 14;                // 槽口宽度
sd_h = 3;                 // 槽口高度

// --- 按钮 ---
btn1_x = 0;               // 按钮 1 X 位置（侧面）
btn1_y = 10;              // 按钮 1 Y 位置
btn_dia = 4;              // 按钮开孔直径

btn2_x = 0;               // 按钮 2 X 位置
btn2_y = 0;               // 按钮 2 Y 位置

// --- 散热孔 ---
vent_dia = 3;             // 散热孔直径
vent_spacing = 8;         // 散热孔间距

// --- PCB 安装孔 ---
pcb_hole_dia = 2.5;       // PCB 安装孔直径
pcb_hole_offset = 3;      // 安装孔距边缘偏移

// --- PCB 固定铜柱高度 ---
pillar_height = 6;        // 铜柱高度（mm）
pillar_dia = 4;           // 铜柱直径

$fn = 64;                 // 圆弧精度

// ============================================================
// 主体模型
// ============================================================

difference() {
    // 外壳主体
    cube([length, width, height], center=true);

    // 内部空腔
    translate([0, 0, wall/2])
        cube([length - 2*wall, width - 2*wall, height - wall], center=true);

    // --- 前面板开孔 ---

    // 激光出光口
    translate([laser_x - length/2, laser_y, 0])
        cylinder(d=laser_dia, h=wall*2+1, center=true);

    // 摄像头窗口
    translate([laser_x - length/2, cam_y, 0])
        cube([cam_w, cam_h, wall*2+1], center=true);

    // ToF 传感器窗口
    translate([tof_x - length/2, tof_y, 0])
        cylinder(d=tof_dia, h=wall*2+1, center=true);

    // --- 后面板开孔 ---

    // 接线出口
    translate([-length/2, cable_y, -height/4])
        cube([wall*2+1, cable_w, cable_h], center=true);

    // --- 侧面开孔（Y 方向两侧） ---

    // PIR 传感器窗口（右侧）
    translate([pir_x, width/2, 0])
        cylinder(d=pir_dia, h=wall*2+1, center=true);

    // SD 卡槽（左侧）
    translate([sd_x, -width/2, sd_y])
        cube([sd_w, wall*2+1, sd_h], center=true);

    // 按钮 1（左侧）
    translate([btn1_x, -width/2, btn1_y])
        cylinder(d=btn_dia, h=wall*2+1, center=true);

    // 按钮 2（左侧）
    translate([btn2_x, -width/2, btn2_y])
        cylinder(d=btn_dia, h=wall*2+1, center=true);

    // --- 顶面散热孔 ---
    for (i = [-2:2]) {
        for (j = [-1:1]) {
            translate([i * vent_spacing, j * vent_spacing, height/2])
                cylinder(d=vent_dia, h=wall*2+1, center=true);
        }
    }

    // --- 底面散热孔 ---
    for (i = [-2:2]) {
        translate([i * vent_spacing, 0, -height/2])
            cylinder(d=vent_dia, h=wall*2+1, center=true);
    }

    // --- PCB 安装孔（底部） ---
    // 四角安装孔
    pcb_l = length - 2*wall - 2*pcb_hole_offset;
    pcb_w = width - 2*wall - 2*pcb_hole_offset;

    for (sx = [-1, 1]) {
        for (sy = [-1, 1]) {
            translate([
                sx * pcb_l/2,
                sy * pcb_w/2,
                -height/2 + wall/2
            ])
            cylinder(d=pcb_hole_dia, h=wall*2, center=true);
        }
    }
}

// ============================================================
// PCB 固定铜柱（可选，打印时取消注释）
// ============================================================
/*
// 四个铜柱
for (sx = [-1, 1]) {
    for (sy = [-1, 1]) {
        translate([
            sx * pcb_l/2,
            sy * pcb_w/2,
            -height/2 + wall + pillar_height/2
        ])
        difference() {
            cylinder(d=pillar_dia, h=pillar_height, center=true);
            cylinder(d=pcb_hole_dia, h=pillar_height+1, center=true);
        }
    }
}
*/
```

### 2.5 打印参数建议

| 参数 | PLA | PETG |
|------|-----|------|
| 层高 | 0.2mm | 0.2mm |
| 填充率 | 20-30% | 20-30% |
| 壳层数 | 3 层 | 3 层 |
| 打印速度 | 50-60 mm/s | 40-50 mm/s |
| 喷嘴温度 | 200-210°C | 230-245°C |
| 热床温度 | 60°C | 70-80°C |
| 是否需要支撑 | 否（设计已考虑） | 否 |
| 冷却风扇 | 100% | 30-50% |

### 2.6 打印后处理

1. 去除支撑材料（如果有）
2. 用砂纸打磨开孔边缘，确保光滑无毛刺
3. 在激光出光口安装安全窗片（见安全设计章节）
4. 在摄像头窗口粘贴透明亚克力片或玻璃片
5. 安装 M2 铜柱和螺丝固定 PCB

## 三、方案 B：亚克力激光切割外壳

### 3.1 概述

亚克力激光切割外壳外观整洁、制作快速，适合有一定动手能力的用户。可以通过淘宝定制切割服务。

### 3.2 材料规格

- 材料：**3mm 黑色透明亚克力板**
- 优点：外观专业，激光切割边缘光滑
- 缺点：开孔位置固定后不易修改，散热较差

### 3.3 板材尺寸

共需要 6 块板材：

| 板材 | 尺寸（mm） | 数量 | 说明 |
|------|-----------|------|------|
| 顶板 | 80 x 60 | 1 | 顶面，带散热孔阵列 |
| 底板 | 80 x 60 | 1 | 底面，带 PCB 安装孔 |
| 前面板 | 60 x 40 | 1 | 激光出光口 + 摄像头窗口 + ToF 窗口 |
| 后面板 | 60 x 40 | 1 | 接线出口 |
| 左侧板 | 80 x 40 | 1 | SD 卡槽 + 按钮 + 散热孔 |
| 右侧板 | 80 x 40 | 1 | PIR 传感器窗口 + 散热孔 |

### 3.4 各板开孔描述

#### 顶板（80 x 60mm）

```
散热孔阵列：5 行 x 3 列，孔径 3mm，间距 8mm
位置：居中分布
```

#### 底板（80 x 60mm）

```

位置：四角各一个，距边缘 5mm
铜柱安装孔：4 个，孔径 4mm（沉头）
```

#### 前面板（60 x 40mm）

```

摄像头窗口：中心偏上，8mm x 8mm 方孔
ToF 传感器窗口：中心偏下偏左，直径 6mm 圆孔
```

#### 后面板（60 x 40mm）

```
接线出口：中心偏下，20mm x 10mm 矩形孔
```

#### 左侧板（80 x 40mm）

```
SD 卡槽：距前端 10mm，14mm x 3mm 槽孔
复位按钮：距前端 10mm，直径 4mm 圆孔
BOOT 按钮：距前端 20mm，直径 4mm 圆孔

```

#### 右侧板（80 x 40mm）

```

```

### 3.5 淘宝定制搜索关键词


### 3.6 组装方式

- 使用 M2 x 8mm 螺丝 + M2 螺母在角部连接各板
- 板材交接处使用 3mm 亚克力角件加固
- PCB 使用 M2 铜柱固定在底板上

## 四、方案 C：万能板/洞洞板简易外壳

### 4.1 概述

最简方案，使用洞洞板和铜柱搭建，适合快速原型验证。不需要任何定制加工。

### 4.2 所需材料

| 材料 | 规格 | 数量 | 参考价格 |
|------|------|------|---------|
| 洞洞板 | 70mm x 50mm，单面 | 2 块 | ¥2 |
| 铜柱（立柱） | M3 x 20mm | 4 根 | ¥1 |
| M3 螺丝 | M3 x 6mm | 8 颗 | ¥1 |
| M3 螺母 | 标准 | 8 颗 | ¥1 |
| 橡胶脚垫 | 自粘式 | 4 个 | ¥1 |

### 4.3 组装步骤

1. **底板**：使用一块洞洞板作为底板
2. **安装铜柱**：在底板四角安装 M3 x 20mm 铜柱
3. **固定 PCB**：将 PCB 放在铜柱上，用螺丝固定
4. **顶板**：将另一块洞洞板放在铜柱顶部，用螺丝固定
5. **开孔**：在顶板对应位置用钻头或烙铁开孔（激光出光口、摄像头窗口等）

### 4.4 优缺点

| 优点 | 缺点 |
|------|------|
| 成本极低（约 ¥6） | 外观简陋 |
| 无需定制加工 | 散热差 |
| 组装快速 | 开孔不精确 |
| 方便修改 | 结构强度低 |

> **注意：** 此方案仅适合原型验证阶段，正式使用建议升级到方案 A 或方案 B。

## 五、外壳组装步骤（通用）

无论选择哪种方案，组装流程基本一致：

### 5.1 准备工作

1. 确认所有元件已焊接完成并通过测试
2. 确认 PCB 尺寸与外壳内部空间匹配
3. 准备固定螺丝、铜柱、螺母

### 5.2 安装顺序

1. **安装 PCB 固定铜柱**：在外壳底板上安装 4 根铜柱
2. **放置 PCB**：将 PCB 对准铜柱位置放入
3. **固定 PCB**：用螺丝将 PCB 固定在铜柱上
4. **连接外部线缆**：连接电源线、传感器线缆等
5. **安装顶板**：盖上顶板，用螺丝固定
6. **安装安全窗片**：在激光出光口安装激光安全窗片
7. **安装摄像头窗片**：在摄像头窗口粘贴透明窗片
8. **最终检查**：确认所有开孔对齐，无遮挡

### 5.3 注意事项

- 安装前确认 PCB 上无短路
- 螺丝不要拧太紧，避免压坏 PCB
- 激光出光口的安全窗片必须安装到位
- 确保摄像头窗口干净无划痕

## 六、散热注意事项

### 6.1 主要发热元件

| 元件 | 典型功耗 | 散热建议 |
|------|---------|---------|
| ESP32-S3-CAM | 300-500mA @ 3.3V | 顶部散热孔 + PCB 铜皮散热 |
| AMS1117-3.3 | 压降 1.7V x 500mA = 0.85W | 加装小散热片或铜柱导热 |
| 振镜驱动板 | 50-100mA @ 5V | 侧面散热孔 |
| MCP4822 + LM324 | < 50mA | 通常不需要额外散热 |

### 6.2 散热优化建议

1. **通风孔设计**：在顶部和侧面设置通风孔，利用自然对流散热
2. **热隔离**：将发热量大的元件（AMS1117）与温度敏感元件（ToF 传感器）分开布置
3. **导热铜柱**：使用铜柱固定 PCB，铜柱可以辅助导热到底板
4. **避免密闭**：不要将外壳完全密封，至少保留 20% 的表面积作为通风孔
5. **环境温度**：避免在环境温度超过 35°C 的地方使用

### 6.3 温度监控

可以通过 ESP32-S3 的内部温度传感器监控芯片温度：

## 七、激光出光口安全设计

### 7.1 安全窗片要求

激光出光口 **必须** 安装激光安全窗片，这是防止意外激光泄露的关键安全措施。

| 参数 | 要求 |
|------|------|
| 光学密度（OD） | OD4+（衰减 10000 倍以上） |
| 波长范围 | 450nm - 650nm（覆盖 RGB 三色激光） |
| 透过率 | 对可见光有一定透过率（方便观察） |
| 材料 | 聚碳酸酯（PC）或专用激光安全玻璃 |
| 安装方式 | 用胶水或压圈固定在出光口 |

### 7.2 安全窗片购买建议

```
淘宝搜索关键词：
- 激光安全窗片 OD4
- 激光防护片 450-650nm
- 激光眼镜片 OD4 全波段
```

> **注意：** 安全窗片的 OD 值必须与激光功率匹配。本项目使用 5mW 激光，OD4 窗片可将泄露激光衰减到 0.5μW 以下，远低于安全限值。

### 7.3 安装方法

1. 将安全窗片裁剪为略大于出光口的圆形（直径约 12mm）
2. 在出光口内侧涂少量 UV 胶或环氧树脂
3. 将窗片压入出光口，确保平整无气泡
4. 等待胶水完全固化
5. 用激光功率计验证窗片的衰减效果

### 7.4 安全窗片维护

- 定期检查窗片是否有裂纹或脱胶
- 如发现窗片损坏，立即更换
- 清洁时使用柔软的无尘布，避免刮花
- 不要使用酒精等溶剂清洁聚碳酸酯窗片
