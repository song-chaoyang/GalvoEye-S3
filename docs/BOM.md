### 运放电路说明

MCP4822 输出 0-2.048V 信号，通过 LM324 运放电路放大并偏移至 ±5V 范围，送入振镜驱动板：

- **X 轴通道**：MCP4822 通道 A → LM324 运放 A → 振镜驱动板 X+
- **Y 轴通道**：MCP4822 通道 B → LM324 运放 B → 振镜驱动板 Y+

> **注意：** LM324 输出 ±5V 需要 ±5V 双电源。如果只有单 5V 电源，输出范围约为 0-3.5V，振镜可能无法满偏。建议使用 DC-DC 模块生成 -5V（见类别六）。

### 激光模组关键说明

1. **三线制（VCC/GND/MOD）**：MOD 引脚接受 TTL/PWM 信号，高电平时激光开启，低电平时关闭。PWM 占空比控制激光亮度。
2. **MOD 引脚限流**：大部分带驱动电路的激光模组 MOD 引脚可以直接接 3.3V 逻辑，不需要额外限流电阻。但如果模组说明书要求串联电阻，请按说明添加（通常 100Ω-1kΩ）。
3. **电平匹配**：ESP32-S3 输出 3.3V，部分激光模组 MOD 引脚需要 5V TTL。如果调制不灵敏，可在 PWM 输出和 MOD 引脚之间加一个电平转换电路（如 BSS138 MOSFET 电平转换模块，约 ¥3）。
4. **安全提示**：5mW Class IIIa 激光对人眼有潜在危害，切勿直视激光束。必须安装激光安全窗片（见类别八）。

## 购买注意事项

### 必须确认的事项

1. **ESP32-S3-CAM 版本**：务必购买 **N8R8 版本**（8MB Flash + 8MB PSRAM），否则无法运行摄像头和神经网络功能。
2. **振镜和驱动板配套**：振镜和驱动板必须配套购买，不同品牌/型号的振镜驱动参数不同，**不能混用**。
3. **激光模组接口**：**必须选带 MOD（TTL 调制）引脚的三线制型号**（VCC/GND/MOD），否则无法通过 PWM 控制亮度。
4. **振镜驱动板输入类型**：**必须选"模拟差分输入"的驱动板**，不要买数字输入的。
5. **安全第一**：务必同时购买安全保护模块（VL53L1X + HC-SR501 + MOSFET），**不要省略安全系统**。
6. **激光安全窗片**：**必须安装**，OD4+ 450-650nm 全波段防护，不可省略。
