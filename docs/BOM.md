## 十、购买注意事项

1. **ESP32-S3-CAM 版本**：务必购买 N8R8 版本（8MB Flash + 8MB PSRAM），否则无法运行摄像头和神经网络功能。
2. **振镜匹配**：振镜和驱动板必须匹配购买，不同品牌/型号的振镜驱动参数不同。
3. **激光模组接口**：确认激光模组的电气接口（电压、接线方式），部分模组需要额外限流电阻。
4. **安全第一**：务必同时购买安全保护模块（VL53L1X + HC-SR501 + MOSFET），不要省略安全系统。
5. **电源余量**：5V 3A 电源已留有余量，但如果同时驱动三个激光模组 + 振镜 + ESP32，建议实测功耗确认。
