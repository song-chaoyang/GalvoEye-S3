# GalvoEye-S3 PCB 修改说明

> 最后更新：2026 年 4 月
> 项目：GalvoEye-S3 — 基于 ESP32-S3 的 DIY 激光投影仪
> 原版 PCB：bbLaser by Corebb

---

## 一、概述

本文档说明如何基于原版 bbLaser PCB 进行修改，以适配 ESP32-S3-CAM 主控板，并新增摄像头、ToF 传感器、PIR 传感器和 MOSFET 激光电源控制等功能。

### 修改目标

1. 将主控芯片从 ESP32（原版）替换为 ESP32-S3-CAM
2. 新增摄像头接口（OV2640，已集成在 ESP32-S3-CAM 上）
3. 新增 I2C 接口，连接 VL53L1X ToF 测距传感器
4. 新增 PIR 接口，连接 HC-SR501 人体红外传感器
5. 新增 MOSFET 接口，控制激光模组总电源
6. 调整 SPI 引脚分配，适配 ESP32-S3 的 GPIO 映射

---

## 二、主要修改点

### 2.1 ESP32 引脚变化

原版 bbLaser 使用标准 ESP32 开发板，引脚分配与 ESP32-S3-CAM 有较大差异。

| 功能 | 原版 ESP32 引脚 | 新版 ESP32-S3-CAM 引脚 | 变化说明 |
|------|----------------|----------------------|---------|
| SPI CS | GPIO5 | GPIO10 | 引脚号变化 |
| SPI CLK | GPIO18 | GPIO12 | 引脚号变化 |
| SPI MOSI | GPIO23 | GPIO11 | 引脚号变化 |
| PWM R | GPIO26 | GPIO4 | 引脚号变化 |
| PWM G | GPIO27 | GPIO5 | 引脚号变化 |
| PWM B | GPIO14 | GPIO6 | 引脚号变化 |
| I2C SDA | GPIO21 | GPIO1 | 新增功能 |
| I2C SCL | GPIO22 | GPIO2 | 新增功能 |
| PIR IN | — | GPIO3 | 新增功能 |
| MOSFET SIG | — | GPIO7 | 新增功能 |

#### PCB 修改操作

1. **删除** 原版 ESP32 底座焊盘区域的走线
2. **重新布线** SPI 信号（CS, CLK, MOSI）到新引脚位置
3. **重新布线** PWM 信号（R, G, B）到新引脚位置
4. **保留** 电源和 GND 走线不变

### 2.2 新增摄像头接口

ESP32-S3-CAM 已集成 OV2640 摄像头，通过 FPC 排线连接。PCB 上需要预留摄像头排线的空间。

#### PCB 修改操作

1. **预留空间**：在 ESP32-S3-CAM 附近预留 20mm x 10mm 的空间，用于摄像头排线弯曲
2. **注意**：摄像头排线方向（正面朝上/朝下），确保排线不被其他元件遮挡
3. **可选**：增加一个摄像头固定支架的安装孔

### 2.3 新增 I2C 接口（ToF 传感器）

新增 VL53L1X ToF 传感器的 I2C 接口。

#### PCB 修改操作

1. **新增焊盘**：在 PCB 边缘区域增加一个 4-pin 排针焊盘
   - Pin 1: 3.3V
   - Pin 2: SDA (GPIO1)
   - Pin 3: SCL (GPIO2)
   - Pin 4: GND
2. **增加上拉电阻**：在 SDA 和 SCL 线上各增加一个 4.7kΩ 上拉电阻到 3.3V
3. **走线**：从 ESP32-S3-CAM 的 GPIO1/GPIO2 走线到 I2C 接口焊盘

### 2.4 新增 PIR 接口

新增 HC-SR501 人体红外传感器的接口。

#### PCB 修改操作

1. **新增焊盘**：增加一个 3-pin 排针焊盘
   - Pin 1: 5V
   - Pin 2: OUT (GPIO3)
   - Pin 3: GND
2. **走线**：从 ESP32-S3-CAM 的 GPIO3 走线到 PIR 接口焊盘
3. **注意**：HC-SR501 输出为 3.3V 高电平，可直接连接 ESP32 GPIO

### 2.5 新增 MOSFET 接口

新增 IRF520 MOSFET 模块的接口，用于控制激光模组总电源。

#### PCB 修改操作

1. **新增焊盘**：增加一个 3-pin 排针焊盘
   - Pin 1: SIG (GPIO7)
   - Pin 2: 5V (VCC)
   - Pin 3: GND
2. **走线**：从 ESP32-S3-CAM 的 GPIO7 走线到 MOSFET 接口焊盘
3. **可选**：在 SIG 线上串联一个 100Ω 电阻，保护 GPIO

### 2.6 其他修改

1. **电源部分**：
   - 保留原有的 5V 和 3.3V 电源电路
   - 确认 AMS1117-3.3 输出电流足够（ESP32-S3 峰值约 300mA）
   - 如需 -5V 电源（LM324 双电源），增加 ICL7660 或电荷泵电路
2. **去耦电容**：
   - 在 ESP32-S3-CAM 的 VCC 引脚附近增加 10μF 电解电容 + 100nF 瓷片电容
   - 在 MCP4822 的 VDD 引脚附近增加 100nF 瓷片电容
3. **板载 LED**：
   - 保留板载状态 LED，连接到 GPIO46（安全状态 LED）

## 三、PCB 布局建议

### 元件布局（从左到右）

```
+------------------------------------------------------------------+
|                                                                  |
|  [ESP32-S3-CAM]    [MCP4822]    [LM324]    [排针接口区]          |
|  (主控+摄像头)      (DAC)       (运放)     (外设连接)            |
|                                                                  |
|  [AMS1117-3.3]     [去耦电容]   [振镜驱动板接口]                 |
|  (稳压)            (C1-C6)     (X+, X-, Y+, Y-)                |
|                                                                  |
|  [I2C 接口]    [PIR 接口]    [MOSFET 接口]                      |
|  (VL53L1X)     (HC-SR501)    (IRF520)                           |
|                                                                  |
|  [电源输入]    [电源指示 LED]                                     |
|  (DC 5V)       (GPIO46)                                         |
|                                                                  |
+------------------------------------------------------------------+
```

### 布线规则

1. **电源线**：至少 0.5mm 宽，优先走内层
2. **信号线**：SPI 和 I2C 信号线 0.25mm 宽，尽量短且直
3. **差分对**：振镜 X/Y 信号线等长、等距走线
4. **地平面**：底层做完整地平面，减少 EMI
5. **隔离**：模拟信号（DAC/运放）和数字信号（ESP32）适当隔离

## 四、面包板/万能板替代方案

如果不想修改 PCB，可以使用面包板或万能板（洞洞板）搭建电路。

### 4.1 面包板方案

**优点**：无需焊接，快速原型验证
**缺点**：接触不良风险，不适合长期使用，高频信号可能不稳定

#### 所需材料

| 材料 | 数量 | 说明 |
|------|------|------|
| 面包板（830 孔） | 1 | 全尺寸面包板 |
| ESP32-S3-CAM | 1 | 通过杜邦线连接 |
| MCP4822 | 1 | 插入面包板 |
| LM324 | 1 | 插入面包板 |
| 阻容元件 | 若干 | 插入面包板 |
| 杜邦线（公对公 + 公对母） | 3 排 | 模块间连接 |

#### 面包板布局

```
面包板分区：

[电源区]          [ESP32-S3-CAM]     [DAC+运放区]       [接口区]
  5V ─────────────────┬──────────────────┬──────────────────
  3.3V ───────────────┤                  │
  GND ────────────────┼──────────────────┼──────────────────
                      │                  │
              [MCP4822]          [LM324]           [排针]
              [阻容元件]          [阻容元件]         [外设]
```

#### 注意事项

1. 面包板的电源轨有中间断点，需要用跳线连通
2. MCP4822 和 LM324 的引脚间距为 2.54mm，可直接插入面包板
3. ESP32-S3-CAM 不能直接插入面包板，需要用杜邦线引出
4. 面包板不适合高频 SPI 信号，如果出现通信错误，尝试降低 SPI 时钟频率

### 4.2 万能板（洞洞板）方案

**优点**：比面包板可靠，可长期使用
**缺点**：需要焊接，走线不如 PCB 整洁

#### 所需材料

| 材料 | 数量 | 说明 |
|------|------|------|
| 万能板（7x9cm） | 1 | 单孔铜箔板 |
| 排针（2.54mm） | 3 排 | 模块接口 |
| 排母（2.54mm） | 2 排 | ESP32-S3-CAM 插座 |
| 焊锡丝 | 1 卷 | 0.6mm 或 0.8mm |
| 飞线 | 若干 | 0.1mm 镀锡铜线 |

#### 焊接步骤

1. **规划布局**：在万能板上用铅笔画出元件位置
2. **焊接排母**：先焊接 ESP32-S3-CAM 的排母插座
3. **焊接 IC 座**：焊接 MCP4822 和 LM324 的 IC 座（DIP 封装可直接焊接）
4. **焊接排针**：焊接各外设接口排针
5. **焊接阻容元件**：按照原理图焊接电阻和电容
6. **飞线连接**：用镀锡铜线连接各信号线
7. **焊接电源线**：最后焊接电源输入和去耦电容

#### 注意事项

1. 万能板上的走线尽量走铜箔面，减少飞线
2. 信号线不要与电源线平行过长，避免干扰
3. 焊接完成后用万用表检查连通性和短路

## 五、立创 EDA 工程链接

PCB 设计文件使用立创 EDA（EasyEDA）编辑。

- **在线编辑器**：https://easyeda.com/editor
- **工程链接**：待创建（将在项目仓库中提供）
- **原版 bbLaser PCB**：参考 Corebb 的原版设计

### 导出文件

从立创 EDA 导出以下文件用于生产：

| 文件 | 格式 | 用途 |
|------|------|------|
| Gerber 文件 | RS-274X | PCB 制造 |
| BOM 文件 | CSV | 元件采购 |
| 坐标文件 | CSV | SMT 贴片 |
| 原理图 PDF | PDF | 文档存档 |
| PCB 渲染图 | PNG | 参考图 |

### 嘉立创打样下单

1. 在立创 EDA 中点击「一键下单 PCB/SMT」
2. 选择 PCB 参数：
   - 层数：2 层
   - 板厚：1.6mm
   - 铜厚：1oz
   - 阻焊：绿油
   - 丝印：白字
   - 表面处理：有铅喷锡（HASL）或 沉金（ENIG）
3. 数量选择 5 片（免费打样）
4. 确认订单并提交

- [ ] PIR 接口连接正确
- [ ] MOSFET 接口连接正确
- [ ] 电源去耦电容已放置
- [ ] 地平面完整

- [ ] Gerber 文件已导出并检查
- [ ] BOM 文件与实际元件一致
