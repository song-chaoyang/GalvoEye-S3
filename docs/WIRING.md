## 二、MCP4822 接线（SPI）

MCP4822 是双通道 12-bit DAC，通过 SPI 接口与 ESP32-S3 通信。

### 引脚对应关系

| MCP4822 引脚 | 功能 | 连接到 ESP32-S3-CAM |
|--------------|------|---------------------|
| VDD (Pin 4) | 电源 (+5V) | 5V 电源 |
| VSS (Pin 5) | 地 (GND) | GND |
| CS (Pin 1) | 片选 | **GPIO10** |
| SCK (Pin 2) | SPI 时钟 | **GPIO12** |
| SDI (Pin 3) | SPI 数据输入 | **GPIO11** |
| VOUTA (Pin 6) | 通道 A 输出 (X 轴) | LM324 运放 A 输入 |
| VOUTB (Pin 7) | 通道 B 输出 (Y 轴) | LM324 运放 B 输入 |
| LDAC (Pin 8) | 同步锁存 | 接 GND（始终更新） |

> **注意：** MCP4822 的通道选择通过 SPI 数据位控制（不需要额外 GPIO 引脚）。LDAC 引脚接 GND 表示每次写入后立即更新输出。

### SPI 通信参数

| 参数 | 值 |
|------|-----|
| SPI 模式 | Mode 0 (CPOL=0, CPHA=0) |
| 时钟频率 | 最高 20 MHz（建议 10 MHz） |
| 数据位宽 | 16 bit（4 位控制 + 12 位数据） |
| CS 极性 | 低电平有效 |

### 接线图示

```
ESP32-S3-CAM                    MCP4822
-----------                    ------
GPIO10 (CS)  ------------------- CS (Pin 1)
GPIO12 (SCK) ------------------- SCK (Pin 2)
GPIO11 (MOSI) ------------------ SDI (Pin 3)
GPIO13 (MISO) --（未使用，悬空）
5V           ------------------- VDD (Pin 4)
GND          ------------------- VSS (Pin 5)
GND          ------------------- LDAC (Pin 8) → 接 GND
```

## 三、LM324 接线（运放电路）

LM324 是四通道运算放大器，用于放大 MCP4822 的 DAC 输出信号，驱动振镜偏转。

### 信号方案说明

MCP4822 的输出范围取决于增益设置：
- 增益 1x（/GA=1）：输出 0 ~ 2.048V
- 增益 2x（/GA=0）：输出 0 ~ 4.096V

振镜驱动板的信号输入要求因型号而异，常见的有两种：
- **差分信号输入**（±5V）：需要运放将 DAC 输出转换为 ±5V 范围
- **单端信号输入**（0-5V）：可以直接使用 DAC 输出，无需运放

### 方案 A（推荐新手）：不使用运放，直接连接

很多入门级振镜驱动板接受 0-5V 单端信号。此时：
- MCP4822 增益设为 2x（固件中已使用 MCP4822_GAIN_2X），输出 0 ~ 4.096V
- X+/Y+ 接 MCP4822 的 VOUTA/VOUTB（Pin 6/Pin 7）
- X-/Y- 接 GND

**优点**：电路最简单，不需要运放、电阻、电容等额外元件
**缺点**：输出范围 0-4.096V，振镜可能无法满偏（满偏需要 0-5V）

### 方案 B（高级）：使用运放放大到 0-5V 范围

如果振镜驱动板需要满量程 0-5V 输入，可以使用同相放大器：

```
MCP4822 VOUTA ---[10kΩ]--- LM324 Pin 3 (同相输入)
                           LM324 Pin 2 (反相输入) ---[10kΩ]--- LM324 Pin 1 (输出)
                                                      |
                                                   [10kΩ] (反馈电阻)
                                                      |
                                                     GND
                           LM324 Pin 1 (输出) ---> 振镜驱动板 X 通道
```

- 增益 = 1 + Rf/Rin = 1 + 10k/10k = 2
- MCP4822 增益 1x 输出 0-2.048V，经运放放大后输出 0-4.096V
- MCP4822 增益 2x 输出 0-4.096V，经运放放大后输出 0-8.192V（会被运放饱和到约 3.5V 单电源范围）

> **注意**：LM324 在单 5V 电源下，输出范围约为 0-3.5V（受限于输出饱和电压）。如果需要输出接近 5V，建议使用轨到轨运放（如 MCP6002、TLV2462）。

### 方案 C（专业）：差分信号输出（需要双电源）

如果振镜驱动板要求差分 ±5V 输入，需要使用反相放大器 + 偏移电路，且 LM324 需要 ±5V 双电源：

```
MCP4822 VOUTA ---[10kΩ]--- LM324 Pin 2 (反相输入)
                           LM324 Pin 3 (同相输入) ---[20kΩ]--- +5V
                                                      |
                                                   [20kΩ]
                                                      |
                                                     GND
                           LM324 Pin 1 (输出) ---[100kΩ]--- Pin 2 (反馈)
                           LM324 Pin 1 (输出) ---[100pF]--- Pin 2 (补偿)
                           LM324 Pin 1 (输出) ---> 振镜驱动板 X 通道
```

- 增益 = -Rf/Rin = -100k/10k = -10
- 偏移电压 = +5V × 20k/(20k+20k) = +2.5V
- 输入 0V → 输出 = 0 × (-10) + 2.5V × (1 + 10) = 27.5V（饱和）
- 输入 1.024V → 输出 = 1.024 × (-10) + 2.5V × 11 = -10.24 + 27.5 = 17.26V（饱和）

> **重要**：上述反相放大器方案增益过大（-10），实际使用时需要根据 MCP4822 的增益设置和振镜驱动板的输入范围重新计算电阻值。建议新手使用方案 A 或方案 B。

### LM324 电源

| LM324 引脚 | 连接 | 说明 |
|-----------|------|------|
| Pin 4 (VCC) | +5V（通过 10μF + 100nF 去耦） | 正电源 |
| Pin 11 (GND) | GND | 负电源（单电源模式） |

> **注意**：单电源模式下，运放输出无法到达负电压。如果需要差分 ±5V 输出（方案 C），Pin 11 需要接 -5V（需要额外的负电源电路）。

### LM324 引脚参考

| 引脚 | 功能 | 本项目使用 |
|------|------|-----------|
| Pin 1 | 1OUT（通道 A 输出） | X 轴信号输出（方案 B/C） |
| Pin 2 | 1IN-（通道 A 反相输入） | 反馈网络 |
| Pin 3 | 1IN+（通道 A 同相输入） | 信号输入 |
| Pin 4 | VCC（正电源） | +5V |
| Pin 5 | 2IN+（通道 B 同相输入） | 信号输入 |
| Pin 6 | 2IN-（通道 B 反相输入） | 反馈网络 |
| Pin 7 | 2OUT（通道 B 输出） | Y 轴信号输出（方案 B/C） |
| Pin 8 | 3IN+（通道 C 同相输入） | 未使用 |
| Pin 9 | 3IN-（通道 C 反相输入） | 未使用 |
| Pin 10 | 3OUT（通道 C 输出） | 未使用 |
| Pin 11 | GND/VEE（负电源） | GND（单电源）或 -5V（双电源） |
| Pin 12 | 4IN+（通道 D 同相输入） | 未使用 |
| Pin 13 | 4IN-（通道 D 反相输入） | 未使用 |
| Pin 14 | 4OUT（通道 D 输出） | 未使用 |

## 四、振镜驱动板接线

### Y 通道

| 驱动板端子 | 连接到 | 说明 |
|-----------|--------|------|

### 驱动板电源

| 驱动板端子 | 连接到 | 说明 |

> **注意：** 不同品牌的振镜驱动板接口定义可能不同，请以实际驱动板说明书为准。部分驱动板使用 DB15 接口。

## 五、RGB 激光模组接线

三个激光模组分别通过 PWM 信号控制亮度。所有激光模组的 VCC 通过 MOSFET 供电（安全控制）。

### 红色激光模组

| 激光模组引脚 | 连接到 | 说明 |
|-------------|--------|------|
| VCC (+) | MOSFET 漏极输出 | 通过 MOSFET 供电 |
| GND (-) | GND | 公共地 |
| MOD (调制) | **GPIO4** (PWM) | 通过限流电阻连接（如需要） |

### 绿色激光模组

| 激光模组引脚 | 连接到 | 说明 |
|-------------|--------|------|
| VCC (+) | MOSFET 漏极输出 | 通过 MOSFET 供电 |
| GND (-) | GND | 公共地 |
| MOD (调制) | **GPIO5** (PWM) | 通过限流电阻连接（如需要） |

### 蓝色激光模组

| 激光模组引脚 | 连接到 | 说明 |
|-------------|--------|------|
| VCC (+) | MOSFET 漏极输出 | 通过 MOSFET 供电 |
| GND (-) | GND | 公共地 |
| MOD (调制) | **GPIO6** (PWM) | 通过限流电阻连接（如需要） |

### PWM 参数

| 参数 | 值 |
|------|-----|
| PWM 频率 | 10 kHz（pin_defs.h 定义：LASER_PWM_FREQ = 10000） |
| PWM 分辨率 | 8 bit (0-255)（pin_defs.h 定义：LASER_PWM_RESOLUTION = 8） |
| 调制方式 | 模拟调制（TTL 电平） |

> **注意：** 部分激光模组的 MOD 引脚需要 5V TTL 电平，而 ESP32-S3 输出 3.3V。如果调制不灵敏，可在 PWM 输出和 MOD 引脚之间加一个电平转换电路（如 BSS138 MOSFET 电平转换模块）。

## 六、VL53L1X 接线（I2C）

VL53L1X 是 ToF 激光测距传感器，用于检测人体距离（第一层安全保护）。

| VL53L1X 引脚 | 连接到 | 说明 |
|-------------|--------|------|
| VCC | 3.3V | 供电 |
| GND | GND | 地 |
| SDA | **GPIO1** | I2C 数据线 |
| SCL | **GPIO2** | I2C 时钟线 |
| XSHUT | 3.3V（或 GPIO） | 传感器复位（上拉使能） |
| GPIO1 | 悬空或接中断引脚 | 中断输出（本项目未使用） |

### I2C 参数

| 参数 | 值 |
|------|-----|
| I2C 地址 | 0x29（默认） |
| 时钟频率 | 400 kHz（标准模式） |
| 上拉电阻 | 4.7kΩ（SDA 和 SCL 各一个，如果模块未自带） |

> **注意：** 如果 I2C 总线上有多个设备，需要确认地址不冲突。VL53L1X 默认地址为 0x29。优先购买板载 4.7kΩ 上拉电阻的模块，如果没有则需要在外部添加上拉电阻（分别接在 GPIO1 到 3.3V 和 GPIO2 到 3.3V 之间）。

## 七、HC-SR501 接线（GPIO）

HC-SR501 是人体红外传感器（PIR），用于检测人体移动（第二层安全保护）。

| HC-SR501 引脚 | 连接到 | 说明 |
|---------------|--------|------|
| VCC | 5V | 供电 |
| GND | GND | 地 |
| OUT | **GPIO3** | 高电平表示检测到人体 |

### 调节参数

| 调节旋钮 | 建议设置 | 说明 |
|---------|---------|------|
| 延时调节（Time Delay） | 最小（逆时针到底） | 检测到人后延时时间，建议最短 |
| 灵敏度调节（Sensitivity） | 中等（中间位置） | 检测距离 3-5m |

> **注意：** HC-SR501 上电后有约 30 秒的稳定时间，期间输出可能不稳定。

## 八、MOSFET 接线（激光电源控制）

IRF520 MOSFET 模块用于控制激光模组的总电源（第三层安全保护）。

| IRF520 模块引脚 | 连接到 | 说明 |
|----------------|--------|------|
| VCC | 5V | 模块供电 |
| GND | GND | 地 |
| SIG (Gate) | **GPIO7** | 控制信号，高电平导通 |
| V+ (Vin) | 5V 电源正极 | 激光电源输入 |
| V- (Load) | 激光模组 VCC | 激光电源输出 |

### 保护电路

```
5V 电源 ---[保险丝 1A]--- MOSFET V+ --- MOSFET V- --- 激光模组 VCC
                                        |
                                     [1N4007]（反向并联，保护 MOSFET）
                                        |
                                       GND
```

> **注意：** 1N4007 反向并联在 MOSFET 输出端，用于吸收激光模组产生的反向感应电动势，保护 MOSFET。

## 九、按钮接线

三个轻触按钮用于用户交互，使用内部上拉电阻（按下时将 GPIO 拉低）。

| 按钮 | 连接到 ESP32-S3-CAM | 另一端 | 功能 |
|------|---------------------|--------|------|
| MODE | **GPIO8** | GND | 模式切换（WiFi/SD 卡播放） |
| PAUSE | **GPIO9** | GND | 暂停/继续播放 |
| NEXT | **GPIO14** | GND | 下一个文件 |

### 按钮参数

| 参数 | 值 |
|------|-----|
| 去抖时间 | 50ms（pin_defs.h 定义：BTN_DEBOUNCE_MS = 50） |
| 长按阈值 | 2000ms（pin_defs.h 定义：BTN_LONG_PRESS_MS = 2000） |
| 上拉方式 | 内部上拉（INPUT_PULLUP） |

> **注意：** 按钮为低电平有效（按下 = LOW）。不需要外部上拉电阻，ESP32-S3 内部上拉已足够。

## 十一、电源分配说明

### 电源架构

```
外部 5V 3A 电源适配器（USB 或 DC）
    |
    +---[AMS1117-3.3 板载]--- 3.3V --- ESP32-S3-CAM (通过 USB 或 5V 引脚)
    |                              |
    |                              +--- VL53L1X (VCC)
    |                              |
    |                              +--- I2C 上拉电阻（如果需要）
    |
    +--- 5V --- MCP4822 (VDD)
    |          |
    |          +--- LM324 (VCC, Pin 4)
    |          |
    |          +--- 振镜驱动板 (V+)
    |          |
    |          +--- HC-SR501 (VCC)
    |          |
    |          +--- IRF520 模块 (VCC)
    |          |
    |          +--- IRF520 V+ (激光电源输入)
    |
    +--- GND --- 所有模块 GND 连接在一起（共地）
```

### 电流预算

| 模块 | 典型电流 | 峰值电流 |
|------|---------|---------|
| ESP32-S3-CAM | 120mA | 300mA（WiFi 发射时） |
| MCP4822 | 1mA | 2mA |
| LM324 | 3mA | 5mA |
| 振镜驱动板 | 200mA | 500mA |
| 红色激光模组 | 30mA | 50mA |
| 绿色激光模组 | 50mA | 80mA |
| 蓝色激光模组 | 40mA | 60mA |
| VL53L1X | 20mA | 40mA |
| HC-SR501 | 65μA | 65μA |
| **总计** | **≈ 465mA** | **≈ 1040mA** |

> 5V 3A 电源（3000mA）完全满足需求，留有充足余量。

## 十三、接线注意事项

1. **先接 GND**：所有模块的 GND 必须连接在一起，确保共地。
2. **电源去耦**：每个 IC 的电源引脚附近放置 100nF 旁路电容。
3. **信号线长度**：SPI 和 I2C 信号线尽量短（< 15cm），避免干扰。
4. **差分信号**：振镜驱动板的 X/Y 信号线使用双绞线或屏蔽线，减少噪声。
5. **激光安全**：接线完成后、上电前，务必确认激光模组未对准人体。
6. **上电顺序**：先接 3.3V（ESP32），再接 5V（外设），最后接激光电源。
7. **引脚一致性**：所有接线必须与 `firmware/src/pin_defs.h` 中的定义一致，否则固件无法正常工作。
