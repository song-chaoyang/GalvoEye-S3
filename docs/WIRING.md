## 二、MCP4822 接线（SPI）

MCP4822 是双通道 12-bit DAC，通过 SPI 接口与 ESP32-S3 通信。

### 引脚对应关系

| MCP4822 引脚 | 功能 | 连接到 ESP32-S3-CAM |
|--------------|------|---------------------|
| VDD | 电源 (+5V) | 5V 电源 |
| VSS | 地 (GND) | GND |
| CS (引脚 2) | 片选 | GPIO1 |
| CLK (引脚 7) | SPI 时钟 | GPIO2 |
| SDI (引脚 4) | SPI 数据输入 | GPIO3 |
| LDAC (引脚 5) | 同步锁存 | GPIO4 |
| VOUTA (引脚 8) | 通道 A 输出 (X 轴) | LM324 运放 A 输入 |
| VOUTB (引脚 6) | 通道 B 输出 (Y 轴) | LM324 运放 B 输入 |
| SHDN (引脚 1) | 关断控制 | 接 VDD（始终使能） |

### SPI 通信参数

| 参数 | 值 |
|------|-----|
| SPI 模式 | Mode 0 (CPOL=0, CPHA=0) |
| 时钟频率 | 最高 20 MHz（建议 10 MHz） |
| 数据位宽 | 16 bit（4 位控制 + 12 位数据） |
| CS 极性 | 低电平有效 |

### 接线图示

```
ESP32-S3-CAM                    MCP4822
-----------                    ------
GPIO1 (CS)   ----[10kΩ 上拉]---- CS (Pin 2)
GPIO2 (CLK)  ------------------- CLK (Pin 7)
GPIO3 (MOSI) ------------------- SDI (Pin 4)
GPIO4 (LDAC) ----[10kΩ 上拉]---- LDAC (Pin 5)
5V           ------------------- VDD (Pin 8)
GND          ------------------- VSS (Pin 5)
             ------------------- SHDN (Pin 1) → 接 VDD
```

## 三、LM324 接线（运放电路）

LM324 是四通道运算放大器，用于将 MCP4822 的 0-2.048V 输出放大并偏移至 ±5V 范围。

### 电路说明

每个通道使用一个运放单元，配置为**反相放大器 + 偏移**：

- MCP4822 输出范围：0 ~ 2.048V
- 运放输出范围：-5V ~ +5V
- 增益：约 -4.88（由反馈电阻决定）
- 偏移电压：+2.5V（由电阻分压提供）

### 通道 A（X 轴）接线

```
MCP4822 VOUTA ---[10kΩ]--- LM324 Pin 2 (反相输入)
                           LM324 Pin 3 (同相输入) ---[20kΩ]--- +5V
                                                      |
                                                   [20kΩ]
                                                      |
                                                     GND
                           LM324 Pin 1 (输出) ---[100kΩ]--- Pin 2 (反馈)
                           LM324 Pin 1 (输出) ---[100pF]--- Pin 2 (补偿)
                           LM324 Pin 1 (输出) ---> 振镜驱动板 X 通道
```

### 通道 B（Y 轴）接线

```
MCP4822 VOUTB ---[10kΩ]--- LM324 Pin 6 (反相输入)
                           LM324 Pin 5 (同相输入) ---[20kΩ]--- +5V
                                                      |
                                                   [20kΩ]
                                                      |
                                                     GND
                           LM324 Pin 7 (输出) ---[100kΩ]--- Pin 6 (反馈)
                           LM324 Pin 7 (输出) ---[100pF]--- Pin 6 (补偿)
                           LM324 Pin 7 (输出) ---> 振镜驱动板 Y 通道
```

### LM324 电源

| LM324 引脚 | 连接 |
|-----------|------|
| Pin 4 (VCC) | +5V（通过 10μF + 100nF 去耦） |
| Pin 11 (GND) | GND |

> **注意：** LM324 需要 ±5V 双电源才能输出 ±5V 范围。如果只有单 5V 电源，输出范围约为 0-3.5V，振镜可能无法满偏。建议使用 DC-DC 升降压模块生成 -5V。

## 四、振镜驱动板接线

振镜驱动板接收模拟差分信号，驱动振镜偏转。

### X 通道

| 驱动板端子 | 连接到 | 说明 |
|-----------|--------|------|
| X+ (差分正) | LM324 Pin 1 输出 | X 轴正信号 |
| X- (差分负) | GND（或反相输出） | X 轴参考地 |
| X_GND | GND | 信号地 |

### Y 通道

| 驱动板端子 | 连接到 | 说明 |
|-----------|--------|------|
| Y+ (差分正) | LM324 Pin 7 输出 | Y 轴正信号 |
| Y- (差分负) | GND（或反相输出） | Y 轴参考地 |
| Y_GND | GND | 信号地 |

### 驱动板电源

| 驱动板端子 | 连接到 | 说明 |
|-----------|--------|------|
| V+ | +5V 电源 | 驱动板正电源 |
| V- | -5V 电源（如有） | 驱动板负电源 |
| GND | GND | 电源地 |

> **注意：** 不同品牌的振镜驱动板接口定义可能不同，请以实际驱动板说明书为准。部分驱动板使用 DB15 接口。

## 五、RGB 激光模组接线

三个激光模组分别通过 PWM 信号控制亮度。

### 红色激光模组

| 激光模组引脚 | 连接到 | 说明 |
|-------------|--------|------|
| VCC (+) | MOSFET 漏极输出 | 通过 MOSFET 供电 |
| GND (-) | GND | 公共地 |
| MOD (调制) | GPIO5 (PWM) | 通过限流电阻连接 |

### 绿色激光模组

| 激光模组引脚 | 连接到 | 说明 |
|-------------|--------|------|
| VCC (+) | MOSFET 漏极输出 | 通过 MOSFET 供电 |
| GND (-) | GND | 公共地 |
| MOD (调制) | GPIO6 (PWM) | 通过限流电阻连接 |

### 蓝色激光模组

| 激光模组引脚 | 连接到 | 说明 |
|-------------|--------|------|
| VCC (+) | MOSFET 漏极输出 | 通过 MOSFET 供电 |
| GND (-) | GND | 公共地 |
| MOD (调制) | GPIO7 (PWM) | 通过限流电阻连接 |

### PWM 参数

| 参数 | 值 |
|------|-----|
| PWM 频率 | 5 kHz（建议） |
| PWM 分辨率 | 8 bit (0-255) |
| 调制方式 | 模拟调制（TTL 电平） |

> **注意：** 部分激光模组的 MOD 引脚需要 5V TTL 电平，而 ESP32-S3 输出 3.3V。如果调制不灵敏，可在 PWM 输出和 MOD 引脚之间加一个电平转换电路（如 BSS138）。

## 六、VL53L1X 接线（I2C）

VL53L1X 是 ToF 激光测距传感器，用于检测人体距离（第一层安全保护）。

| VL53L1X 引脚 | 连接到 | 说明 |
|-------------|--------|------|
| VCC | 3.3V | 供电 |
| GND | GND | 地 |
| SDA | GPIO11 | I2C 数据线 |
| SCL | GPIO12 | I2C 时钟线 |
| XSHUT | 3.3V（或 GPIO） | 传感器复位（上拉使能） |
| GPIO1 | 悬空或接中断引脚 | 中断输出（本项目未使用） |

### I2C 参数

| 参数 | 值 |
|------|-----|
| I2C 地址 | 0x29（默认） |
| 时钟频率 | 400 kHz（标准模式） |
| 上拉电阻 | 4.7kΩ（SDA 和 SCL 各一个） |

> **注意：** 如果 I2C 总线上有多个设备，需要确认地址不冲突。VL53L1X 默认地址为 0x29。

## 七、HC-SR501 接线（GPIO）

HC-SR501 是人体红外传感器（PIR），用于检测人体移动（第二层安全保护）。

| HC-SR501 引脚 | 连接到 | 说明 |
|---------------|--------|------|
| VCC | 5V | 供电 |
| GND | GND | 地 |
| OUT | GPIO13 | 高电平表示检测到人体 |

### 调节参数

| 调节旋钮 | 建议设置 | 说明 |
|---------|---------|------|
| 延时调节（Time Delay） | 最小（逆时针到底） | 检测到人后延时时间，建议最短 |
| 灵敏度调节（Sensitivity） | 中等（中间位置） | 检测距离 3-5m |

> **注意：** HC-SR501 上电后有约 30 秒的稳定时间，期间输出可能不稳定。

## 八、MOSFET 接线（激光电源控制）

IRF520 MOSFET 模块用于控制激光模组的总电源（第三层安全保护）。

| IRF520 模块引脚 | 连接到 | 说明 |
|----------------|--------|------|
| VCC | 5V | 模块供电 |
| GND | GND | 地 |
| SIG (Gate) | GPIO14 | 控制信号，高电平导通 |
| V+ (Vin) | 5V 电源正极 | 激光电源输入 |
| V- (Load) | 激光模组 VCC | 激光电源输出 |

### 保护电路

```
5V 电源 ---[保险丝 1A]--- MOSFET V+ --- MOSFET V- --- 激光模组 VCC
                                        |
                                     [1N4007]（反向并联，保护 MOSFET）
                                        |
                                       GND
```

> **注意：** 1N4007 反向并联在 MOSFET 输出端，用于吸收激光模组产生的反向感应电动势，保护 MOSFET。

## 九、电源分配说明

### 电源架构

```
外部 5V 3A 电源适配器
    |
    +---[AMS1117-3.3]--- 3.3V --- ESP32-S3-CAM (VCC)
    |                          |
    |                          +--- VL53L1X (VCC)
    |                          |
    |                          +--- I2C 上拉电阻
    |
    +--- 5V --- MCP4822 (VDD)
    |          |
    |          +--- LM324 (VCC, Pin 4)
    |          |
    |          +--- 振镜驱动板 (V+)
    |          |
    |          +--- HC-SR501 (VCC)
    |          |
    |          +--- IRF520 模块 (VCC)
    |          |
    |          +--- IRF520 V+ (激光电源输入)
    |
    +--- GND --- 所有模块 GND 连接在一起（共地）
```

### 电流预算

| 模块 | 典型电流 | 峰值电流 |
|------|---------|---------|
| ESP32-S3-CAM | 120mA | 300mA（WiFi 发射时） |
| MCP4822 | 1mA | 2mA |
| LM324 | 3mA | 5mA |
| 振镜驱动板 | 200mA | 500mA |
| 红色激光模组 | 30mA | 50mA |
| 绿色激光模组 | 50mA | 80mA |
| 蓝色激光模组 | 40mA | 60mA |
| VL53L1X | 20mA | 40mA |
| HC-SR501 | 65μA | 65μA |
| **总计** | **≈ 465mA** | **≈ 1040mA** |

> 5V 3A 电源（3000mA）完全满足需求，留有充足余量。

## 十一、接线注意事项

1. **先接 GND**：所有模块的 GND 必须连接在一起，确保共地。
2. **电源去耦**：每个 IC 的电源引脚附近放置 100nF 旁路电容。
3. **信号线长度**：SPI 和 I2C 信号线尽量短（< 15cm），避免干扰。
4. **差分信号**：振镜驱动板的 X/Y 信号线使用双绞线或屏蔽线，减少噪声。
5. **激光安全**：接线完成后、上电前，务必确认激光模组未对准人体。
6. **上电顺序**：先接 3.3V（ESP32），再接 5V（外设），最后接激光电源。
